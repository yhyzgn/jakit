## 1、插件引入

> 此处以`maven`仓库`pom`引入方式为例
>

```xml
<dependency>
  <groupId>com.yhyzgn.jakit</groupId>
  <artifactId>util</artifactId>
  <version>${latestVersion}</version>
</dependency>
```
